repeat()

repeat() 메서드는 문자열을 복사하여, 복사한 새로운 문자열을 반환합니다. 

const str1 = "javascript";
const currentStr1 = str1.repeat(0);     //''
const currentStr2 = str1.repeat(1);     //javascript
const currentStr3 = str1.repeat(2);     //javascript javascript
