01. 다음의 결과값을 작성하시오.

{
    function func(){
        for( i =1; i<=7; i++){

02. 다음의 결과값을 작성하시오.

{
    function func(){
        let i = 10, hap = 0;
        while(i>1){
            i--;
            if( i % 3 ==1){
                hap += i;
            }
        }
        document.write(hap);
    }
    func();
}

i--로 9부터 시작하고 if문안에 들어갈 수 있는 i는 9~1이며, i를 3으로 나눠 나머지값이 1이 나오는 것은 7,4,1가 되고 합은 12.

04. 다음의 결과값을 작성하시오.

{
    function func(){
        let a = [1,2,3,4,5,6,7,8,9], cnt = 0;

        for( i=0; i<a.length; i++){
            if(a[i] % 2 == 0){
                cnt = cnt + 1;
            }
        }
        document.write(cnt);
    }
    func();
}

i<9 인 a값을 if 조건문에 대입하면 짝수의 합을 구하는 것이 되며, cnt는 0이고 조건에 맞을때마다 +1 하여, 2,4,6,8나와서 4개가 됩니다.

정답 확인하기

05. 다음의 결과값을 작성하시오.

{
    function func(data,exp){
        let result = 1;
        for( i = 0; i<exp; i++){
            result = result * data
        }
        document.write(result);
    }
    func(3, 3);
}

func에 3,3을 그대로 대입시켜보면 for문 i에 들어갈 수 있는 숫자는 0,1,2가 됩니다. 0부터 돌립니다. 시작하며, result는 1인 상태이고 1*3 = 3이 됩니다. 다시 위로 올라가서 1을 하고, 2를하면, 1*3= 3 , 3*3 = 9 , 9*3 = 27.

정답 확인하기
27

07. 다음의 결과값을 작성하시오.

{
    function func(){
        let i, j = 0;
        for( i = 0; i<5; i++){
            j+= i;
        }
        document.write(i);
        document.write(j);
    }
    func();
}

i에 들어갈 수 있는 숫자가 0,1,2,3,4
j는 계속 더한다. i가 0일때 0, 0+1=1, 1+2=3, 3+3=6, 6+4=10. 5는 for문을 만족시키지 못하니 반복문을 빠져나오게 되고, 답은 i는 5 , j는 10. (for 밖에 i의 출력이기에)

09. 다음의 결과값을 작성하시오.

{
    function func(){
        let a = 0, sum = 0;
        while( a < 10){
            a++;
            if(a % 2 ==1) continue;
            sum += a;
        }
        document.write(sum);
    }
    func();
}

while 을 지나서 a++를 받고나면 a는 1~10. if문로 2를나눈 나머지가 1인 숫자는 1,3,5,7,9. continue로 앞에서 구한 값들 외의 값으로 진행하여, 짝수의 합이 됩니다. 2+4+6+8+10으로 30

19. 다음의 결과값을 작성하시오.

{
    function solution(a, b){
        return (a + b) * ((a > b ? a - b : b - a) + 1) / 2;
    }
    document.write(solution(5, 9))
}

a > b 이 아니기에 ?는 if와 같기에, b - a로 계산. 14 * 5 / 2 = 35

정답 확인하기
35

20. 다음의 결과값을 작성하시오.

{
    function solution(text){
        return text.split('').join('/').replace('s','site')
    }
    document.write(solution('webs'))
}

text에 webs넣고 split('')으로 쪼내고 join(/)을 실행하면 w/e/b/s가 되고, replace로 s를 site로 바꿔해주면 w/e/b/site.
