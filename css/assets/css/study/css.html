CSS 단위

글자 크기를 명시하지 않을 경우 16px 기본 값.


px

해상도에 따라 상대적으로 달라지는 기본 단위입니다.

font-size: 12px;


%

부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위입니다.

font-size: 150%;

특별한 설정이 없다면 16px이 100%가 되므로, 150%는 24px이 됩니다. 만약 부모 요소가 10px이었다면 150%는 15px이 될 것입니다.


em

부모 요소의 글자 100% 기준으로 계산한 크기를 100 분의 1단위입니다.

font-size: 1.5em;

특별한 설정이 없다면 16px이 1em이 됩니다.


rem

최상의 요소의 크기를 100% 기준으로 계산합니다.
즉 root, html 요소를 뜻하므로, html에서 지정한 글자 크기가 1rem 이 됨니다

font-size: 1.5rem;


vw, vh

vw는 뷰포트 너비값의 100분의 1단위입니다.
vh는 뷰포트 높이값의 100분의 1단위입니다.

font-size: 10vw;

뷰포트 너비의 10% 크기라는 의미.
